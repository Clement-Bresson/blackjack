# blackjack
A React blackjack game for 1 player (for now)

## Technologies
  - create-react-app (React)
  - Redux
  - styled-components

## How to play (quick)
  1 - Clone this repository  
  2 - `cd [repository-where-you-cloned-it]`  
  3 - `npm install`  
  4 - `npm start` 

## How to publish
  1 - Clone this repository  
  2 - `cd [repository-where-you-cloned-it]`  
  3 - `npm install`  
  4 - `npm run build` 
  5 - `npm install -g serve` 
  6 - `serve -s build`
